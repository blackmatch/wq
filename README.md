# wq

[![NPM version][npm-image]][npm-url]

Weather Query Command Line Tool in China.查询国内天气情况的终端工具。

## 特性

* 实时从中央气象台获取数据。
* 简单易用。
* 丰富多彩的输出结果。

## 安装

```shell
sudo npm install wq-cli -g
```

## 使用

```shell
Usage: wq <name>

Options:
  -V, --version  output the version number
  -h, --help     output usage information
```

示例：

```shell
wq 深圳
```
